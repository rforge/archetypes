Archetypal Analysis

R package

The main function archetypes implements a framework for archetypal analysis supporting arbitary problem solving mechanisms for the different conceputal parts of the algorithm.

Download

The stable version of archetypes is available on CRAN; issue the following from within R to install it: 

R> install.packages("archetypes")

The development version is available on R-Forge.

Demonstration

Check the vignette; e.g., 

R> vignette("archetypes", package = "archetypes")
